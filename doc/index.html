Description
An Scons library which wraps interface of various compilers in to one universal.
With Nucleotide it is easy to write Scons build files avoiding to explicitly state platform or compiler.

Quick example:
Next line set RTL:
settings.accumulate( 'RTL', { 'type': 'static', 'configuration' : 'debug' } )
and replace:
if( os.platform == 'win32' ):
    if( compiler == 'msvc' ):
        I_flag = 'M'
        if( 'dynamic' == retrieve_RTL_type() ):
            I_flag += 'D'
        if( 'static' == retrieve_RTL_type() ):
            I_flag += 'T'
        if( 'debug' == retrieve_configuration_type() ):
            I_flag += 'd'
        if( 'release' == retrieve_configuration_type() ):
            pass
        env.append( CPPFLAGS, [ I_flag ] )

    if( compiler == 'cygwin' ):
        env.append( LINKFLAGS, "-static" )

    if( compiler == 'mingw' ):
        env.append( LINKFLAGS, "-static" )
else:
    if( os.platform == 'linux' ):
        if( compiler == 'gcc' ):
            env.append( LINKFLAGS, "-static" )
