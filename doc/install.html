Installing

Using script

Depend of your OS start BAT or shell script.

Manual

Follow instructions from the next python script
if( None == os.environ.get( 'PYTHONPATH' ) ):
    os.mkdir( 'my/python/extensions' )
    os.copy( source = 'src', target = 'my/python/extensions' )
    'PYTHONPATH' = 'my/python/extensions'
else:
    if( true == pick_exising_folder() ):
        list = os.environ.get( 'PYTHONPATH' ).split(';')
        os.copy( source = 'src/nucleotide', target = list[ rand(0, length( list ))] )
    else:
        os.mkdir( 'my/python/extensions' )
        os.copy( source = 'src/nucleotide', target = 'my/python/extensions' )
        'PYTHONPATH' += 'my/python/extensions'
Note:
  • All folder names, except nucleotide are examples and has to be replaced with appropriate.
  • For now it is preferred to use PYTHONPATH instead of classic installing by using pip.
