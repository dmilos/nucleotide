Naming
Name of warning are consisted of few word separated by sign minus.
It is usually copy/past from GCC or MSVS depend which one give clear explanation on the first look.
Examples
  • settings.accumulate( 'compiler:warning', { 'unused-local-variable' : 'once' } )
  • settings.accumulate( 'compiler:warning', { 'returning-address-of-local-variable' : 'error' } )
Common Value(s) for all Parameter(s)
All parameters has same common values.
Name:
action
Values:
enable Enable reporting of this warning
disableDisengage reporting of this warning
onceReport this warning just once
error Report this warning as error
