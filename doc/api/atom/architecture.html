Atom Architecture

Description
Set architecture of target file(s)
Example(s)
  • settings.accumulate( 'architecture' , { 'name' : 'x86_64' } )
Parameters
name
Type
string
Description
State the name of architecture.
Value(s):
x86_64, i586, x86, x86_amd64, i486, emt64, i386, ia64, itanium, amd64, i686
Note
This list is taken from Scons. In some cases this may not have effect.
