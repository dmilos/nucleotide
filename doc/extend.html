New compiler

General rule is follow the pattern of GCC or MSVC, they are created in same manner.

Here is step by step explanation.
  1. make new folder with <OS> name in ./component if not exists
  2. make new folder with compiler name in ./component<OS>
  3. create file translator.py and __init__.py
  4. Copy content of some existing compiler e.g. GCC or MSVC
  5. make folder atom in ./component<OS>/<new-compiler>
  6. Create __init__.py in ./component<OS>/<new-compiler>/atom
  7. Populate folder atom in same manner as some exiting e.g. GCC or MSVC
